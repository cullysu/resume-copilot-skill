## Master Resume Markdown

```markdown
# Name
City | email | phone | LinkedIn | GitHub/portfolio

## Summary
2-3 lines targeted to the role. Mention domain, strongest tools, and evidence-backed impact.

## Skills
- Category: skill, skill, skill
- Category: skill, skill, skill

## Experience
### Role, Organization
Dates | Location
- Action + scope + tool/method + result.
- Action + complexity + measurable or observable impact.

## Projects
### Project Name
Dates | Link
- Problem + build/action + stack + result.

## Education
Degree, School | Dates
- Optional honors, coursework, awards, thesis, activities.
```

## Reactive Resume Field Mapping

Use this after drafting content.

```markdown
Basics:
- Name:
- Headline:
- Email/phone/location:
- URL profiles:

Summary:
- Paste targeted summary.

Sections:
- Experience -> work history entries.
- Projects -> project entries, not work history unless they were paid/professional.
- Education -> schools/degrees.
- Skills -> grouped categories if the template supports it; otherwise concise skill lists.
- Custom section -> awards, certificates, publications, open source, patents, volunteering.
```
